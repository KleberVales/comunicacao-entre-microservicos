# comunicacao-entre-microservicos

### Comunicação entre microserviços

```css

 ├─ Síncrona
 │   ├─ REST
 │   ├─ GraphQL
 │   └─ gRPC
 │
 └─ Assíncrona
     ├─ Mensageria / Eventos
     │   ├─ RabbitMQ
     │   ├─ Kafka
     │   └─ SQS
     │
     └─ Outros
         ├─ Webhooks
         └─ Callbacks
```
